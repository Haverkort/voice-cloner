# Voice Cloner

Try it live here: https://voice-cloner.vercel.app

## Setup

Clone this repo, make a [Replicate](https://replicate.com) account and update the `.env` file with your API token:

```
REPLICATE_API_TOKEN=<token>
```

## Run it

```bash
npm install
npm run dev
```
